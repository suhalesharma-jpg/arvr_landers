🏛 AR Smart Guide

An Augmented Reality based interactive monument exploration system

📌 Project Overview

AR Smart Guide is an interactive Augmented Reality application designed to enhance monument and museum exploration. It uses real-time image recognition to identify artifacts and display contextual historical information through engaging AR overlays.

🚀 Key Features

  • Real-time image recognition using MindAR
  • Interactive AR overlays
  • Historical background & fun facts
  • Audio narration support
  • Structured development documentation

🛠 Tech Stack

  • HTML, CSS, JavaScript
  • Three.js
  • MindAR
  • GitHub Pages Deployment
